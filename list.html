1)What is HTML?
HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web.
Key Points:
  • HyperText refers to links (text or images) that connect to other web pages.
  • Markup Language means HTML uses tags to define the structure and formatting of content
2)What is difference between HTML and XHTML?
The main difference between HTML and XHTML lies in how strictly they follow rules. Here's a clear comparison:
HTML (HyperText Markup Language)
  • More flexible and forgiving.
  • Designed to be readable and work even if the code is not perfect.
  • Not strict about tag usage or closing tags
XHTML (eXtensible HyperText Markup Language)
  • Stricter and more XML-based version of HTML.
  • All tags must be properly closed, nested correctly, and lowercase.
  • Browsers are less forgiving of mistakes.

4)What is Element In Html?
  • An HTML element is a complete unit in an HTML document that defines how content should be displayed in a web browser. It typically consists of a start tag, some content, and an end tag
  • Some elements, called self-closing or empty elements do not have closing tags.
  • Elements are the building blocks of webpages, and they help structure and format text, images, links, tables, and other content on a site.
5)Difference between html4 and HTML5?
HTML4
HTML4 is the older version of the language used to make websites. It only allowed basic page structure, and if you wanted to add things like videos, audio, or animations, you had to use other tools like Flash or extra scripts.
HTML5
HTML5 is the newer version that makes it easier to build modern websites. It lets you add videos, audio, games, and animations directly using simple tags. It also includes new tags to make your code more organized. Plus, HTML5 works better on phones, tablets, and modern browsers.
